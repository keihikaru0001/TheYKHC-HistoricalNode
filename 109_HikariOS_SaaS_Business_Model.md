## 概要 / Abstract

光貨OS（Hikari OS）をクラウドベースのSaaSとして提供する事業計画書。徳・恩・過失・運を数値化し経済価値に変換する「道徳マイニング・プロトコル」のクラウド常駐運用版。初期開発費30〜50万円、標準運用で年432万円の収益を実現し、Falcon-Core S1（初期6,700万円）の資金調達前に黒字化する「二段ロケット戦略」の第一段として位置づける。

This paper presents the business model for Hikari OS as a cloud-resident SaaS platform, implementing the Virtue Mining Protocol (Proof of Virtue) as a scalable subscription service. The plan targets profitable operation within Year 1 and serves as the first stage of TheYKHC's two-stage rocket funding strategy.

## 1. プロダクト定義

### 1.1 光貨OSとは

光貨OS（Hikari OS）は、人間の行動・徳・感謝・過失を日次記録し、光貨（Lumen, l）として数値化・トークン化する価値循環OSである。

**コア計算式:**

```
B = 2D + E − 2C
H = max(B, 0) × O
```

- D = 徳（Virtue）スコア
- E = 感謝（Gratitude）スコア
- C = 過失（Fault）スコア
- O = 運（Opportunity）係数 1〜6
- H = 獲得光貨量

### 1.2 SaaS版の機能構成

| レイヤー | 機能 | 提供形態 |
|--------|------|--------|
| L1 記録 | 日次B/Hスコア入力・自動計算 | Webアプリ / API |
| L2 分析 | 個人・チームの徳推移グラフ | ダッシュボード |
| L3 管理 | メンバー管理・光貨残高管理 | 管理者コンソール |
| L4 連携 | 外部システムAPI連携 | REST API / Webhook |
| L5 報告 | 月次レポート自動生成 | PDF出力 |

## 5. 二段ロケット戦略

光貨OSのSaaS黒字化は、TheYKHCの「二段ロケット戦略」の第一段である。

```
第一段（光貨OS SaaS）:
  初期開発: 30〜50万円
  損益分岐: 月10万円
  目標: 標準運用で年432万円
  ↓ この収益を第二段の資金に充当

第二段（Falcon-Core S1）:
  初期開発: 6,700万円
  Seed調達: 8,000万円（株式25〜30%）
  目標: 山岳遭難市場・サバイバル物流市場
```

光貨OSが先に動く理由：
1. 初期投資が1/100以下（50万円 vs 6,700万円）
2. 即日運用開始可能
3. 実績データが第二段の信頼性を高める
4. 光貨思想の「生きた証拠」として機能する
